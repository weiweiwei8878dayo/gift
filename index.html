Frequently asked questions

Who is the code generator for?

It's dedicated for people running marketing campaigns, using random promo codes. For example, you can use it to generate promo codes, gift card codes, referral codes, loyalty ids, and more. No duplicates guaranteed.

How random are the codes?

The longer pattern the better.

Let’s take 8 characters long strings for example:charset = 0123456789 abcdefghijkl mnopqrstuvwxyz ABCDEFGHIJKLM NOPQRSTUVWXYZ — 63 characters. In this case there are 63⁸ = 248155780267521 possible codes. It means that if you issue a billion codes the probability of guessing a code is 10⁹/63⁸ = 0.000004… — 4 in a million.

What is the maximum number of codes I can generate?

The maximum number of generated codes in one session is 9999.

What generated codes can be used for?

Generated codes can be used as promo codes, voucher numbers, gift cards numbers, referral codes, serial numbers, lottery numbers, strong passwords, and more. If you need help with running promotional campaigns, you should give Voucherify a try.

What is code prefix?

The code prefix is the first few characters of code. It enables you to assign codes to a particular marketing campaign.

What’s the code generator?

Voucherify code generator is a free online coupon maker for generating unique random codes using letters, digits and other characters.

What is code postfix?

The code postfix is the last few characters of code. Useful for campaign reporting.

What’s a code pattern?

Code pattern is the way the code is made. You can define a pattern using following characters: uppercase, lowercase, digits, or special characters. You can also decide about the sequence.

Can I customize the length, prefix and pattern of codes?

Yes, these random code parameters are customizable.

Which characters can codes contain?

Codes can mix upper cases, lower cases, digits, special characters, punctuation and brackets.

Is registration required to create codes?

No, you can use the code generator without any registration. Note that Voucherify Promotion Engine comes with a free 30-day trial and a free plan, too.

What is the maximum length for generated codes?

The maximum code length is 100 characters.

Is payment required to create codes?

No, the voucher creator is free.

Can I save generated codes as CSV file?

Yes, you can copy and paste it to Excel and then save as a CSV file. You can also export the generated codes as a CSV file straight away.

I need more codes that you can generate. What should I do?

You can change one of the used parameters (e.g. code length) and run the tool again. Remember to use the Clear button to reset code settings.

Do you store the generated codes?

No, we don’t store the codes after they have been generated and we don’t have a list of pre-generated codes to choose them from. They are generated at the requested time.
